A 1.7.10 mod that doubles the inventory size in the most seamless way possible.

You can move from each inventory page by either clicking on the bar between the inventory and hotbar or by using the scrollwheel.
The second inventory can be accessed in other containers like chests, furnances, etc... 
It also works with modded containers if they access the inventory. (This feature seems to work well with simple containers but more complex ones will likely need patches)
